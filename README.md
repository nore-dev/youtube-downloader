# Youtube Video/Audio Downloader

Worst youtube downloader

![screenshot1](ss1.png)
![screenshot2](ss2.png)

### Install
```
npm install -g ytdljs
```

### Examples
```
ytdljs // no args

ytdljs https://www.youtube.com/watch?v=OWzmmCBTJZ0 -q 360p -o test

ytdljs OWzmmCBTJZ0  -o test2 -d onlyVideo
```
